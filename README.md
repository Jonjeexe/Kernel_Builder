### Kernel Builder
Script to compile kernel 
```cmd
curl -O https://raw.githubusercontent.com/Jonjeexe/Kernel_Builder/refs/heads/main/Compiler.sh
```

### Edit Compiler.sh
1. add host name 
2. add user name

### Edit codename
change codename_deconfig to ur own codename for example
blossom_deconfig

### Start Compile
enter your kernel source then 
```cmd
bash Compiler.sh
```
